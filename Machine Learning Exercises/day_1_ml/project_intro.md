# Digital Marketing Conversion Analysis — Day 1 ML Project

## Project Goal

The goal of this project is to apply Day 1 Machine Learning concepts using a real digital marketing dataset.

The dataset contains customer information, campaign information, engagement behavior, and conversion-related results.

I will use this project to understand how different marketing and customer features relate to conversion performance.

## Main Learning Goal

I want to learn Linear Regression step by step.

The main regression target will be:

`ConversionRate`

This is suitable for regression because it is a continuous numeric value.

## Important Note

The dataset also contains a column called:

`Conversion`

This is a binary column:

- `0` = did not convert
- `1` = converted

This is better suited for classification, not normal linear regression.

So in this notebook:

1. I will first use Linear Regression for `ConversionRate`.
2. Later, I may use Logistic Regression for `Conversion`.
