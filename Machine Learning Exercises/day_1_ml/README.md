# Machine Learning Day 1

This repository contains my Day 1 Machine Learning practice work.

## Project

Digital Marketing Conversion Analysis using Linear Regression.

## Dataset

Dataset used from Kaggle:

`rabieelkharoua/predict-conversion-in-digital-marketing-dataset`

The notebook downloads the dataset using KaggleHub.

## Topics Covered

- Data loading with KaggleHub
- Data inspection
- Data quality checks
- Exploratory data analysis
- Basic statistics
- Simple Linear Regression
- Residuals and Least Squares
- Multiple Linear Regression
- Train/test split
- One-hot encoding
- Model evaluation
- Business interpretation

## Main Conclusion

Linear Regression helped explain the modeling process, but it was not a strong model for predicting `ConversionRate` in this dataset.

The analysis showed that the model mostly predicted close to the average conversion rate, and the R² values were very low.
